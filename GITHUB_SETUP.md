# GitHub Setup

Recommended repository name:

`scsm2223-ch13-books-fullstack`

## Create the repository

1. Create a new empty GitHub repository with the name above.
2. Do not add a README, license, or `.gitignore` on GitHub.
3. Connect this local repo to the new remote.

## Push

```bat
git remote add origin https://github.com/WONGZIQI0212/scsm2223-ch13-books-fullstack.git
git push -u origin main
```

## After pushing

- Keep using `main`.
- Push each new lab change as a normal commit.
